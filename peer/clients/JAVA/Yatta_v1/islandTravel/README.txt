islandTravel
============
This involves the fun stuff. Talking over the wire, sending files, real time streaming, you name it.
The benefit here is the protocol consolidation for JXTA (Advertise), Gnutella(Query,Search), Gnutella2, 
Bittorrent, Exosee, Svn, Git, Nbd, eDonkey2000/eMule, Ftp, RTP, MMS, IGMP, ICMP, SOAP,and Http (streams,POST, GET). 
The feature of course is the new Yatta framework (using TCPIP/UDP)
